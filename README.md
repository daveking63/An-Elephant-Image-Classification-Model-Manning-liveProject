# An-Elephant-Image-Classification-Model-Manning-liveProject
A project in Manning Publications 'liveProject' program. In this case the overall task is to develop a straightforward CNN classification model for determining whether a single computer (optical) image contains an African Elephant or an Asian Elephant. Technically speaking, this 'liveProject' is accomplished by creating the following series of five interrelated Jupyter (Python) Notebooks including:

<ol>
  <li>Curate Image Dataset</li>
  <li>Preprocess the Data</li>
  <li>Design and Build the (Python CNN) Model</li>
  <li>Train and Evaluate the Model</li>
  <li>Iterate for Improvement</li
</ol>
